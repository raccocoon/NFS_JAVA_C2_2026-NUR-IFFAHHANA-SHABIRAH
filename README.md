## Full-Stack Development with Java, React & MongoDB

Full-Stack Development with Java, React & MongoDB is a 20-day instructor-led programme covering web fundamentals, Java, Spring Boot, React, and MongoDB. It progresses from basic programming concepts to full-stack application development including REST APIs, authentication, database modelling, testing, and capstone delivery. AI tools are used for scaffolding, refactoring, testing, and query design, but all outputs must be verified and owned by the learner.

## Programme Objective & Goal

```
- Understand web fundamentals, HTTP, REST, and JSON
- Write basic to intermediate Java and JavaScript
- Build REST APIs using Spring Boot
- Apply authentication, authorisation, and validation
- Design MongoDB data models and queries
- Build React frontend applications: routing and state management
- Implement testing for backend and frontend systems
- Develop a complete capstone full-stack application
- Use AI tools responsibly for learning and development support
```
## How to Navigate This Repository

Each day is organized into its own folder using the format `dayX`, ***X = day number*** containing:

- `src/` → Source code of any programming language implemented
- `exercises/` → Written tasks and answers
- `Day_0X_Progress-Log-Summary.md`
